¿Cómo usar WalkieTalkieParaNenes?

1. Carga completamente el WalkieTalkieParaNenes antes de usarlo por primera vez.

2. Enciende el dispositivo presionando el botón de encendido durante 3 segundos.

3. Selecciona el mismo canal en ambos dispositivos para que puedan comunicarse entre sí.

4. Presiona y mantén presionado el botón de hablar cuando quieras enviar un mensaje.

5. Suelta el botón de hablar para escuchar las respuestas del otro dispositivo.
