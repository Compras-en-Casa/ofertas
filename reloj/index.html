¿Cómo usar reloj?

1. Ajusta la correa del reloj para que se ajuste cómodamente a tu muñeca.

2. Asegúrate de que el reloj esté en la hora correcta girando la corona o usando los botones según el modelo.

3. Si tu reloj tiene funciones adicionales, como cronómetro o alarma, consulta el manual para aprender a configurarlas.

4. Para relojes automáticos, asegúrate de darles cuerda si no los has usado en un tiempo.

5. Limpia el reloj regularmente con un paño suave para mantenerlo en buen estado.
